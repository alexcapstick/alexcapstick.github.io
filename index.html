Alex Capstick

About Me

I am a PhD student at Imperial College London and my research is mainly focused on using machine learning techniques for the extraction of clinical insights from in-home monitoring data. Within healthcare my work has mostly revolved around people living with Dementia.

My interests are in theoretical machine learning and building new techniques that are applicable to the world of healthcare.

My Interests

I am curious about all things ML and Maths, but specifically:

I have worked in the following areas:

  • Meta-learning
  • Continual learning
  • Anomaly detection
  • Learning from unreliable data
  • Self-supervised learning
  • Generative modeling
  • The use of pretrained models in healthcare monitoring

Projects

The following are some projects I am working on:

    This is a module I am a assistant teacher for at Imperial College London for the MSc Computational Neuroscience. I also put the course website together, which is a work in progress!

  • AML Python Package

    Some of my Machine Learning code.

  • AVT Python Package

    Some of my graphing code, inspired by seaborn's API.

  • SKU Python Package

    Sci-kit Learn Utils add extra functionality to pipelines, and allow for more complex machine learning training and testing. It also includes some extra functionality for hyper-parameter optimisation.

  • Dcarte-Transform Python Package

    Dcarte-Transform package adds extra functionality to dcarte, an in-house python package designed for loading the UK DRI Minder data. Specifically Dcarte-Transform adds extra abilities for machine learning research. It also includes a feature engineering recipe for easy access to engineered features.

  • PDUtils Python Package

    Pandas Utils add extra functionality to DataFrames.
